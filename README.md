# 🏦 Bank Management System

A **Console-Based Bank Management System** developed in **C++** using **Object-Oriented Programming (OOP)** concepts. The application simulates the core functionalities of a banking system, allowing users to create and manage accounts, perform financial transactions, and maintain customer records with persistent file storage.

This project demonstrates practical implementation of OOP principles, file handling, and data management while providing a user-friendly console interface.

---

## 📌 Features

### 👤 Account Management

* Create new bank accounts
* Update customer information
* Delete existing accounts
* Search customer accounts
* View account details

### 💰 Banking Operations

* Deposit money
* Withdraw money
* Check account balance
* Maintain transaction records

### 📂 Data Management

* Persistent data storage using CSV files
* Secure record management
* Easy retrieval and updating of customer information

## 🚀 Getting Started

### Prerequisites

* C++ Compiler (MSVC, G++, MinGW)
* Visual Studio or Visual Studio Code

### Clone the Repository

```bash
git clone https://github.com/your-username/bank-management-system.git
```

### Compile

```bash
g++ app.cpp -o bank-management-system
```

### Run

```bash
./bank-management-system
```
